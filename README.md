# Food Delivery Analytics SQL Project

## Project Overview

This project analyzes a food delivery platform database using MySQL to generate insights related to customers, restaurants, orders, payments, and deliveries.

The project demonstrates practical SQL skills through real-world business analytics and reporting.

---

## Database Schema

The database consists of five interconnected tables:

- Customers
- Restaurants
- Orders
- Payments
- Delivery

These tables are connected using primary and foreign key relationships to simulate a real-world food delivery platform.
